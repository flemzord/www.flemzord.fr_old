This is a simple Jekyll theme, that is derived from the one I use on my blog. You can find out more info about customizing your Jekyll theme, as well as basic Jekyll usage documentation at : jekyllrb.com

You can find the source code for this theme at : https://github.com/jbleuzen/sheekyll

Please don't criticize the code much, initially it was just a quick and dirty mockup and it ended as the theme I use for almost 2 years now. But as two people asked if could share it, it landed on Github.
